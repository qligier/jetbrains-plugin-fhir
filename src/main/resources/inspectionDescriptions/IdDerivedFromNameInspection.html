Reports ids that aren't based on the item's name.

If the item has a name, the id SHOULD be based on the item's name, with '_' replaced by '-', changed to lowercase, and truncated if necessary. The overall length MUST NOT be more than 64 characters (per the requirements of the FHIR id datatype).

See https://build.fhir.org/ig/HL7/fhir-shorthand/reference.html#item-identifiers for more information.
